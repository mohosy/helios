# Helios

A browser-based microgrid optimization console that simulates 24-hour demand/solar dynamics and battery dispatch strategy.

## Highlights
- Synthetic demand, solar, and price signal generation.
- Battery dispatch optimizer for peak shaving and cost reduction.
- Stress-test mode for volatile pricing and reduced generation scenarios.
- Visual timeline for demand, solar, grid import, and discharge behavior.

## Stack
- HTML5 Canvas
- Vanilla JavaScript
- CSS dashboard UI

## Run
Open `index.html` in your browser.
